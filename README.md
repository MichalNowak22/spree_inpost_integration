SpreeInpostIntegration
======================

Spree integration with inPost Api.


Installation
----------------

Add gem to `Gemfile`:

```console
gem 'spree_inpost_integration', github: 'MichalNowak22/spree_inpost_integration'
```

Install gem:
```console
rails g spree_inpost_integration:install
```
